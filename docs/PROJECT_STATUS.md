## מצב הבדיקות (ריצה אחרונה: 2026-07-05)

- `npm test` — **69/69 עוברות**
- `npm run typecheck` — נקי (renderer + electron)
- `npm run lint` — נקי (כולל react-hooks recommended)
- `npm run build` — תקין (vite + tsc)
- `npm run dist:win` — תקין; הופקו:
  - `release/Super Mastermind 1.0.0.exe` (portable, ~71MB)
  - `release/Super Mastermind Setup 1.0.0.exe` (מתקין NSIS)
- בדיקת עשן לגרסה הארוזה — האפליקציה נפתחת עם כותרת עברית תקינה
