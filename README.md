# בול פגיעה — Super Mastermind

משחק מחשב מלא בעברית המבוסס על Mastermind (בול פגיעה): אפליקציית Desktop
עצמאית ל־Windows, עם מנוע משחק גמיש, ניתוח לוגי בזמן אמת, מצב „ניחוש מכריע”,
בניית חידות ידניות, סטטיסטיקות והיסטוריה — הכול מקומי, ללא צורך באינטרנט.

## תכונות מרכזיות

- **משחק קלאסי** — המחשב מגריל רצף סודי; מנחשים ומקבלים בולים (מקום נכון)
  ופגיעות (צבע נכון, מקום שגוי), עם חישוב מתמטי מדויק גם בצבעים חוזרים.
- **חלונית ניתוח לוגי** — כמה רצפים עדיין אפשריים, כמה נפסלו בניחוש האחרון,
  האם קיים פתרון יחיד, והאם אפשר לפתור בוודאות בניסיונות שנותרו.
- **מצב „ניחוש מכריע”** — מקבלים היסטוריית ניחושים מוכנה וניסיון אחד בלבד
  למצוא את הסוד. 4 דרגות קושי, כולל מצב פתרון־יחיד מובטח.
- **חידה ידנית** — בונים חידה בעצמכם, בודקים עקביות, מאתרים את שורת הסתירה,
  רואים את הפתרונות, שומרים ומייצאים/מייבאים JSON.
- **הגדרות מלאות** — אורך רצף, מספר צבעים, כפילויות, ניסיונות, הגבלת זמן,
  רמזים, המלצות (כולל מינימקס בהשראת Knuth), עורך צבעים מלא, תבניות קושי.
- **נגישות** — RTL מלא, סמלים בנוסף לצבעים (עיוורון צבעים), טקסט לכל ציון,
  קיצורי מקלדת (1–9, Enter, Backspace, Delete), ערכות נושא וגודל ממשק.
- **פרטיות** — הכול נשמר מקומית בלבד; אפס תקשורת רשת.

## דרישות מערכת

- Windows 10/11 (להרצת הגרסה הארוזה)
- לפיתוח: Node.js 20+ ו־npm

## התקנה והרצה (משתמש קצה)

הורידו מהתיקייה `release` (אחרי בנייה) אחד מהשניים:

- `Super Mastermind Setup <version>.exe` — מתקין רגיל (NSIS)
- `Super Mastermind <version>.exe` — גרסה ניידת ללא התקנה

הקבצים אינם חתומים — SmartScreen עשוי להזהיר; בחרו "More info → Run anyway".

## פיתוח

```bash
npm install        # התקנת תלויות
npm run dev        # Vite + Electron עם טעינה חמה
npm run dev:vite   # שרת Vite בלבד (לפיתוח UI בדפדפן)
```

## בדיקות ואיכות

```bash
npm test           # 69 בדיקות יחידה (Vitest)
npm run typecheck  # TypeScript strict — renderer + electron
npm run lint       # ESLint + react-hooks
```

## בנייה ואריזה

```bash
npm run build      # typecheck + vite build + קומפילציית electron
npm run dist:win   # EXE נייד + מתקין NSIS לתוך release/
npm run dist:dir   # תיקיית win-unpacked בלבד (מהיר, לבדיקות)
```

## מבנה הפרויקט

```text
electron/             תהליך ראשי של Electron (TypeScript)
src/game-engine/      חוקי המשחק וחישוב ציונים
src/solver/           יצירה וסינון פתרונות, ניתוח, המלצות, רמזים
src/challenge-generator/  מחולל חידות „ניחוש מכריע”
src/settings/         ברירות מחדל, תבניות קושי ואימות
src/storage/          שמירה מקומית (הגדרות, סטטיסטיקות, היסטוריה, חידות)
src/workers/          Web Worker לחישובים כבדים
src/components/       רכיבי ממשק
src/screens/          מסכי האפליקציה
src/types/            טיפוסי TypeScript משותפים
src/utils/            כלי עזר
src/tests/            בדיקות אוטומטיות
docs/                 תיעוד מלא
```
